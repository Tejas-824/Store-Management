## 🚀 Getting Started

### 📦 Backend Setup

1. Navigate to backend folder:

```bash
cd D:\Your Folder name\store-management\backend
```

2. Run database migrations:

```bash
node src\db\migrate.js
```

3. Run seed data:

```bash
node src\db\seeds\seed.js
```

4. Start backend server:

```bash
npm run dev
```

---

### 💻 Frontend Setup

1. Navigate to frontend folder:

```bash
cd D:\Your Folder name\store-management\frontend
```

2. Start frontend:

```bash
npm run dev
```

---

## ⚡ Notes

* Make sure `.env` is properly configured before running. Copy the contents from .env.example
* Backend should be running before starting frontend
* Default frontend runs on: http://localhost:5173
